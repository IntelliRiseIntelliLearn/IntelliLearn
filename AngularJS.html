Introduction To AngularJS

Posted By:ABC

Angular.js is an open source JavaScript MVC (Model View Controller) framework developed by Google. It provides user a highly structured approach to develop applications. It provides user with the features reducing 10 lines of JavaScript code to just a single line which reduce development time and leads to very high productivity.

Single Page Application (SPA)


A Single Page Application (SPA) is that which has different shell (i.e. sections) on single page in which different views are loaded.

Normally in a traditional app if there is need to load only particular section but typically it loaded everything again on a blink.

In SPA first the upfront (i.e. Initial content) is loaded later we load the different views or webpages can say into the different shell.
