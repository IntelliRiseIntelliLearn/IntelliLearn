Use CDN (Content delivery Network)

Advantages of Using CDN


Some browser allows a fix no of concurrent file downloading from single server, if you are using your own server then it may delay the downloading of script file from the server and if CDN is used then the concurrent downloading will not encounter any problem. If any other site opened on client PC have used Angular.js using CDN the browser will be having the same in cache and will not re-download, saving data and page load time

Downloading Script File


To download the script file goto http://angularjs.org



Hit the download button and following page opens:



Select the options as required by you and download the file.
Now include the following code in your application head tag to include th downloaded file
<script type=”text/javascript” src=”path of angular.min.js” ></script>
Now you are ready to use Angular in your project.
