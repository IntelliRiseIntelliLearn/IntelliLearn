Data Binding

There are two ways of data binding MVC: - A beautiful concept of Model View Controller is trending these days and is a very efficient way of processing data.
Let’s have an overview:
Model: - It communicates with data provider such as Database of application.
It receives request from controller and provides the required data.
View: - It receives data from model, process it and finally display.
Controller: - A mediator between View and Model. It receives data from Model, manipulate it and passes the data to View.
Routing: - It is basically the concept of SPAs. It is the concept of processing views on the shell in SPAs through built in routing support.

Testing

JqLite


jqLite is mini version on JQuery which is built in Angular.js
It helps in DOM (Document Object Model) manipulation.
The no. of JQuery functions available ale less, If user wants to use more, can play with JQuery together.

Directives


Directives is the concept used by Angular for performing different function on DOM elements. Directives control the functionality. Eg:- ng-app – It tells angular to start and show its magic.

Validation


Angular also has built in functionality of form validation which reduces the code to higher extent and can be implemented at an ease.

Dynamic Injection


Dynamic Injection is the injection of data in a DOM element dynamically without loading the page again, saving data.
